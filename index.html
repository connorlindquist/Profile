Sample of projects I've worked on

Shopping With Friends

Social shopping app build in my Objects and Design class. I developed it using Agile software development with eleven one week springs. The app is built to facilitate more interaction among friends when looking for deals on specific items.

16-bit ASM

Mathematical sequence evaluator for LC3 processor that performs mathematical operations ranging from addition, and subtraction to exclusive or and bitshifts. Built using Logism.

Mario GBA

Gameboy Advance game in Mode 3. Collect coins and mushrooms while avoiding bombs to increase speed and points. Implements collision detection as well as DMA for efficiency.

C Malloc and Free

C implementation of malloc, free, realloc, and calloc. Free of memory leaks with thorough error handling. Freeing of memory using buddy system from 2048 block.
